# T1 shop

[T1 Shop 바로가기](https://t1-shop.netlify.app/)
<br/>
관리자가 상품을 등록, 수정할 수 있는 기능을 구현했습니다.
<br/>

- React Hook 사용
- React Router로 페이지 이동
- Firebase의 Authentication으로 로그인
- Firebase의 RealtimeDB에 데이터 저장
- Cloudinary에 이미지 파일 저장
